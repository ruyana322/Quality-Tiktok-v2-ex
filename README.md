## 📌 Tentang

**Quality Video TikTok** adalah Chrome Extension yang memungkinkan video TikTok kamu tampil dalam kualitas **1080p 60fps** dengan cara menginjeksi sinyal kualitas ke dalam file video sebelum diupload.

Tanpa proses patch ini, TikTok akan otomatis mengkompresi video kamu dan tidak dapat ditampilkan dalam kualitas HD.

## ✅ Persyaratan

Sebelum memulai, pastikan kamu sudah menyiapkan:

| Kebutuhan | Keterangan |
|-----------|-----------|
| 🌐 Browser | Chrome / Kiwi Browser (Android) |
| 🔒 VPN | **Kingdom VPN** (sangat disarankan) |
| 📱 Video | File MP4 siap upload (resolusi 1080p) |
| 🌍 Server VPN | Region US / SG |

## 📦 Instalasi Extension

### Android (Kiwi Browser)

1. Download file `.zip` extension dari halaman **[Releases](../../releases)**
2. install file ZIP di penyimpanan
3. Buka **Kiwi Browser**
4. Ketuk menu **⋮** → **Extensions**
5. Aktifkan **Developer Mode** (pojok kanan atas)
6. Ketuk **Load unpacked (+ from .zip)**
7. Pilih folder hasil dowload tadi
8. Extension **Quality Video TikTok** akan muncul di daftar → aktifkan

### Desktop (Chrome)

1. Download dan extract file `.zip` extension
2. Buka `chrome://extensions`
3. Aktifkan **Developer Mode**
4. Klik **Load unpacked** → pilih folder hasil extract
5. Extension siap digunakan

## 🚀 Cara Penggunaan

Ikuti langkah-langkah berikut **secara berurutan** untuk mendapatkan hasil 1080p:

---

### Langkah 1 — Aktifkan VPN

```
Buka Kingdom VPN → pilih server US atau SG → Connect
```

Pastikan VPN sudah **terhubung** sebelum melanjutkan ke langkah berikutnya.

---

### Langkah 2 — Z-Payload Inject

1. Tap ikon extension di toolbar browser
2. Tap tombol **Pilih Video** → pilih file MP4 kamu
3. Tap tombol **⚡ Z-PAYLOAD INJECT**
4. File hasil patch akan **otomatis terunduh** dengan nama `[nama]_patched.mp4`

---

### Langkah 3 — MTLib Patch

1. Tap tombol **Pilih Video** kembali
2. Pilih file hasil unduhan dari Langkah 2 (`_patched.mp4`)
3. Tap tombol **🔧 MTLIB PATCHER**
4. File final akan **otomatis terunduh** — ini file yang siap diupload

---

### Langkah 4 — Upload ke TikTok

1. Buka **[tiktok.com/upload](https://www.tiktok.com/upload)** atau **TikTok Studio**
2. Upload file hasil dari Langkah 3
3. Isi caption, hashtag, dll
4. **Set ke Private** sebelum posting
5. Tap **Posting / Publish**

---

### Langkah 5 — Verifikasi & Publish

1. Tunggu proses upload dan processing TikTok selesai
2. Buka video kamu → tap **⋮** → **Tinjau** untuk cek kualitas
3. Jika kualitas sudah **HD / 1080p** → ubah ke **Public**
4. Tunggu **2–5 menit** setelah dipublish
5. ✅ Video kamu sekarang tampil **1080p 60fps**

## ❓ FAQ

**Q: Kenapa harus pakai VPN?**
> TikTok membatasi kualitas video berdasarkan region. Dengan VPN ke US/SG, server TikTok yang menangani upload kamu mendukung resolusi 1080p.

**Q: Kenapa harus dua kali patch?**
> Z-Payload dan MTLib adalah dua proses berbeda yang saling melengkapi. Z-Payload mempersiapkan struktur video, MTLib menanamkan sinyal kualitas yang dikenali TikTok.

**Q: Berapa lama proses patch?**
> Sangat cepat — proses berjalan langsung di browser kamu, tidak ada file yang dikirim ke server. Biasanya selesai dalam hitungan detik.

**Q: Apakah aman?**
> Ya. Proses patch berjalan 100% di browser (client-side). Tidak ada data yang dikirim ke server manapun.

**Q: Video tetap 720p setelah semua langkah?**
> Pastikan VPN aktif dan menggunakan region US/SG. Coba juga tunggu beberapa menit setelah publish sebelum mengecek kualitas.
